## Technologies
Python and Rembg Library

## How to Use

`OS: Windows 10`

`Python Version: 3.9.5`

`git clone https://github.com/canerucal/remove-bg.git`

`cd remove-bg`

`python3 -m venv env`

`source env/bin/activate`

`pip install -r requirements.txt`

`run the code in terminal`
